# 增删改查语法

## 1.插入记录

- （1）常规方法
  `insert into 表名（字段 1， 字段 2， 字段 3，...）values
(值 1， 值 2， 值 3，...);`
- (2) 简化方式
  `insert into 表名 values
(值 1， 值 2， 值 3，...);`
  > 注意：若省略字段，则对所有字段按照定义顺序进行赋值。
- （3）一次插入多条记录
  ` insert into 表名 values
(值 1， 值 2， 值 3，...)，
(值 1， 值 2， 值 3，...)，
(值 1， 值 2， 值 3，...)，
(值 1， 值 2， 值 3，...)，
....;`

## 2. 更新数据

`update 表名
set 字段 1=值 1，字段 2=值 2，字段 3=值 3，....
where 条件；`

> 注意：若省略条件，则更新所有的记录

## 3.删除数据

`delete from 表名
where 条件；`

> 注意：若省略条件，则删除所有的记录

## 4.查询记录

- select
- from
- where
- group by
- having
- order by
- limit
